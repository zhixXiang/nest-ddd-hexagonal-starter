# nest-ddd-hexagonal-starter

## Description

A hexagonal starter architecture implemented using NestJS and references Get Your Hands Dirty on Clean Architecture.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Then you can access from http://localhost:3000/accounts/1

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
